Instrument recognition

Banjo or violin ?

Description of the application :

This application has been designed for children aged 4 to 7 years old who would like to learn about music and instruments.

The application shows whether the children are able to recognise the instruments.

This application could be useful for teachers in primary schools or in music schools for an introduction to instruments.

In our example we will focus on the banjo and the violin, this application can be extended to all instruments.

Image classification using MobileNet and p5.js
